# 工数ログ

顧問先別に工数を記録する、縦型タイムラインの単一HTMLアプリ。外部依存なし・LocalStorage保存。

公開URL: https://forestmtk.github.io/time-tracker/

## 主な機能
- タイムラインで工数を記録（開始/停止、後からの手入力・修正）
- 記録の複製（同じ顧問先・業務・長さを直後にワンタップ配置）
- 締め忘れ検知（前日以前の計測が止まっていなければ起動時に締め直しを促す）
- 集計（日/月の切替。顧問先別・業務別の合計、月は稼働日数とコピーあり）
- 日報の書き出し（Markdown / テキスト。コピー、またはObsidianフォルダへ直接保存）

## データ
- 保存先: ブラウザのLocalStorage（キー `timeTracker.v1`）。端末・ブラウザ単位で、サーバには送信しない。
- バックアップ: 設定からJSONを書き出し／読み込み（`time_tracker_backup_YYYY-MM-DD.json`）。
- 端末を移す・ブラウザを変える場合は、このJSONで移行する。

## Obsidian 直接保存（Chrome / Edge のみ）
- 初回に書き出しシートで保存先フォルダを一度選ぶ（例: Googleドライブ同期の Obsidian 仕事用フォルダ）。
- 以降は「Obsidianに保存」ボタン、または翌日起動時の自動保存（許可が残っていれば無操作、要求されたらバナーの「保存」を1回）。
- 日報は `YYYY-MM-DD.md` に `<!-- time-tracker -->`〜`<!-- /time-tracker -->` のブロックとして書き込む。手書き部分は保持され、再保存でブロックだけ差し替わる。
- iPhone / Safari はファイル書き込み非対応のため、コピーして貼り付け運用。

## メモ
- 単一HTMLファイル（`index.html`）。ビルド不要、そのまま開けば動く。
- 変更は `index.html` を上書きするだけ。URL・LocalStorageのデータは据え置き。
